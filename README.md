# 🏥 HealthViz AI

## 📌 Description
HealthViz AI is a smart healthcare data visualization system that allows users to upload CSV data, visualize it, generate insights, and predict disease risks.

## 🚀 Features
- Upload healthcare dataset
- Data visualization (bar, pie, histogram)
- AI-based risk prediction
- Automated insights
- NLP query system

## 🛠 Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run app.py
